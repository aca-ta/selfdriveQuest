# SelfdriveQuest

道路をたくさん走らせて、賢い自動運転を作ろう！

深層強化学習（DQN）を使った自動運転AIをブラウザ上で体験できるインタラクティブな学習アプリです。自分でコースを設計し、AIを訓練して、その実力を試すことができます。

## 遊び方

### 1. コースを設計する

グリッド上にクリックで道路を配置し、オリジナルのコースを作成します。スタートとゴールをドラッグで配置でき、グリッドサイズは 5x5 / 10x10 / 15x15 から選べます。ランダム生成も可能です。3つ以上のコースを作成して学習に使いましょう。

### 2. AIを訓練する

作成したコースでDQNエージェントを訓練します。ハイパーパラメータ（学習率・割引率・探索率など）を調整でき、プリセットも用意されています。訓練の進捗はリアルタイムでチャートとログに表示されます。

### 3. 実力を試す

訓練済みAIを未知のコース10問でテストします。成功率・経路効率（BFS最短経路との比較）からスコアが算出されます。失敗したコースを訓練データに追加して再学習させることもできます。プレイグラウンドモードでは自作コースでAIの動きを観察できます。

## 技術スタック

- **React 19** + **TypeScript** - UI
- **TensorFlow.js** - ブラウザ内でのDQN学習・推論
- **Web Worker** - 訓練処理をバックグラウンドスレッドで実行（UIをブロックしない）
- **Vite** - ビルドツール
